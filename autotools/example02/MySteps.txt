1. I create a c++ file in src directory.

2. Now I need to create a file configure.ac, for that I will use first the autoscan command
2.1. autoscan  inside my project root directory

$ autoscan

This create a file called configure.scan
$ cat configure.scan
#                                               -*- Autoconf -*-
# Process this file with autoconf to produce a configure script.

AC_PREREQ([2.69])
AC_INIT([FULL-PACKAGE-NAME], [VERSION], [BUG-REPORT-ADDRESS])
AC_CONFIG_SRCDIR([src/example_queue.cpp])
AC_CONFIG_HEADERS([config.h])

# Checks for programs.
AC_PROG_CXX

# Checks for libraries.

# Checks for header files.

# Checks for typedefs, structures, and compiler characteristics.

# Checks for library functions.

AC_OUTPUT

2.2. Copy or change the name of the autogenerated file and change the values in AC_INIT and add 
AM_INIT_AUTOMAKE([-Wall -Werror foreign])
$ cp configure.scan configure.ac
